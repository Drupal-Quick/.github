Quick is a Composer-based scaffolding tool for Drupal. Point it at a config.dq.yml file and it provisions a site, generates a themed front-end, and applies recipes — from a blank DDEV install to a deployed static site in a few commands. Quick then removes all references to itself and provides a blank slate for continued development. See our [workflow docs](https://github.com/Drupal-Quick/drupal-quick/blob/main/docs/workflow.md).
